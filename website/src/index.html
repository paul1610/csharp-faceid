How to install [code]

Simply pull the project from the repository and start the Visual Studio .sln file. After the loading is done, switch the ouput next to the start to csharpFaceId.

How to install [zip]

Simply download the .zip program from the linked website and start it by just unziping & following the [code] steps

How to use

Now a simple interface should pop up, with a live preview of your camera. Just press "Take photo" and wait for the result on the right side.
